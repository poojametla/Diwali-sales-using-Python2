# Diwali-sales-using-Python2
Data Analysis using python


Analyzed Diwali sales data to identify customer purchasing behavior and sales trends using Python.

Performed data cleaning and preprocessing (handling null values, data type conversion) using Pandas and NumPy.

Conducted exploratory data analysis (EDA) to find insights based on gender, age group, state, occupation, and product categories.

Created visualizations using Matplotlib and Seaborn to support business decision-making.
